
# Personal Expense Manager

## Project Description 

This is a simple python command-line application to track personal expenses.Users can add expenses, view expenses ,and generate basic reports like total,category - wise ,average ,and monthly expenses .

## Technologies Used 

- Python 
- CSV File Handling 
- Object - Oriented Programming 

## Project Structure 

PERSONAL_FINANCE_MANAGER /
|
|- main.py             #Entry point of the application 
|- expenses.py         # Expense Class 
|- expense_manager.py  # Manages Expenses 
|- file_manager.py     # Handles CSV read/write 
|- reports.py          # Generate Reports 
|- utils.py            #Validation and Helper functions
|- data /
    |__expenses.csv     # Stores expense data 




## How to Run the Project 

1. Open terminal in project folder 

2. Run :
    python main.py 

3.Follow Menu options to add or view expenses 


## FEATURES 

- Add New expenses 
- View All Expenses
- Total Expense calculation 
- Category wise expense summary 
- Average expense calculation 
-  Monthly Expense Report
