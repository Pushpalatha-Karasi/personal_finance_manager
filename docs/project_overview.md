## Introduction
The Personal Finance Manager is a command-line based Python application designed to help users track and analyze their daily expenses.

The application allows users to record expenses, categorize them, analyze totals, generate monthly reports, and safely store data using CSV files.

## Objectives
- To understand Python object-oriented programming
- To implement file handling using CSV
- To practice error handling and input validation
- To design a modular and scalable application
- To simulate a real-world finance tracking system

## Scope of the Project
This project is intended for educational purposes and demonstrates how a small-scale finance management system can be built using Python without external frameworks.

## Target Users
- Beginners learning Python
- Students practicing data handling and reporting
- Individuals wanting a simple expense tracker
