## System Requirements
- Python 3.10 or above
- Any operating system (Windows, Linux, macOS)

## Installation Steps

1. Download or clone the project repository
2. Open the project folder in VS Code or terminal
3. Run the application using:
   python main.py

## Application Menu

1. Add Expense
2. View Expenses
3. View Total Expense
4. Category-wise Expense
5. Average Expense
6. Monthly Reports
7. Filter Expenses by Category
8. Backup Data
9. Exit

## How to Add an Expense
- Enter date in YYYY-MM-DD format
- Enter category (auto-cleaned by system)
- Enter a valid amount (positive number)
- Enter description

## Backup Feature
Whenever a new expense is added, the system automatically creates a backup of the CSV file in the backup folder.
